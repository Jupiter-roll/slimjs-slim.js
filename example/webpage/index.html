Web components plus Magic!

With es6 and slim.js you could write less code. Native web components developed double-time with no hassle, no need for external libraries that affect your workflow. When it comes to development with web components, slim.js is your new best friend.

Featuring

Data binding

<span bind>Hello, [[myProperty]] </span>
...
this.myProperty = 'slim.js'

Attribute binding

<greeter greeter-attribute="[[myProperty]]">Hello, slim <greeter/>
...
this.myProperty = 'someValue'

Method binding

<greeter greeter-attribute="[[myMethod(myProperty)]]">Hello, slim <greeter/>
...
myMethod(prop) { return something }

Repeaters

<greeter slim-repeat="people" bind>Hello, [[data.name]] <greeter/>
...
this.people = [ {name:'John'}, {name:'Jenny'}, {name: 'Jeremy'} ... ]

Extensible / Plugins

Slim.plugin('beforeRender', function(element) { do something } )

Get slim.js

npm install slim-js
bower install slimjs
