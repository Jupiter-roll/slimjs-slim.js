A better way for creating NATIVE web components

With es6 and slim you could write native web components double-time with no hassle of using external libraries that affect your workflow

Featuring

Text binding

<span bind>Hello, [[myProperty]] />
...
this.myProperty = 'slim.js'

Attribute binding

<greeter my-attribute="[[myProperty]]">Hello, slim <greeter/>
...
this.myProperty = 'someValue'

Method binding

<greeter my-attribute="[[myMethod(myProperty)]]">Hello, slim <greeter/>
...
myMethod(prop) { return something }

Repeaters

<greeter slim-repeat="people" bind>Hello, [[data.name]] <greeter/>
...
this.people = [ {name:'John'}, {name:'Jenny'}, {name: 'Jeremy'} ... ]

Extensible / Plugins

Slim.plugin('beforeRender', function(element) { do something } )

Better lifecycle

onBeforeCreated / onCreated / beforeRender / render / afterRender / update ...

Get slim.js

npm install slim-js
bower install slimjs
